NewKeysUtil changes keys in property file and replace keys in java files of selected project.
Step 1: Make copy of proeperty file and project folder;
Step 2: Run NewKeysUtil.jar;
Step 3: Choose property file  you are going to change using File Chooser Window, press 'Open';
Step 4: If you are going to rewrite excisting file, select 'Yes' at dialog window. If 'No', you will be suggested to create new property file;
Step 5: Choose project directory using Directory Chooser Window, press 'Change';
Step 6: Check that everything's done properly in property file and project folder.
